# Случайная величина

- **Определение** ::
  - Величина, принимающая различные числовые значения в зависимости от исхода случайного эксперимента.

- **Среднеквадратическое отклонение (СКО)** ::
  - Квадратный корень из дисперсии.

- **Единицы измерения СКО** ::
  - Те же, что и у самой случайной величины.
  - Пример: если рассматривается напряжение шума на выходе радиоприёмного устройства (РПрУ), то СКО шума измеряется в вольтах.

- **Зачем вводить СКО, если есть дисперсия?** ::
  - Дисперсия имеет размерность [СВ]², а СКО — размерность самой случайной величины → СКО удобнее интерпретировать физически и сравнивать с другими величинами.

- **Основные свойства дисперсии** ::
  - Прибавление константы к случайной величине не меняет дисперсию.
  - При умножении случайной величины на константу c дисперсия умножается на c².
  - Если X и Y независимы → D(X + Y) = D(X) + D(Y).

- **Пример из радиотехники** ::
  - Дисперсия шума определяет минимальный обнаруживаемый сигнал.
  - Чем больше дисперсия → тем труднее выделить полезный сигнал на фоне шума.

- **Момент** (в математической статистике) ::
  - Числовая характеристика распределения случайной величины.

- **Корреляция** ::
  - Мера статистической взаимосвязи между двумя объектами (величинами, процессами, сигналами), показывающая, насколько изменение одного сопровождается изменением другого.

- **На какой главный вопрос отвечает корреляция?** ::
  - Насколько один объект «похож» на другой или насколько сильно связаны их изменения.

- **Физическая трактовка корреляции** ::
  - Мера перекрытия двух функций в пространстве их значений.

- **Геометрический / сигнальный смысл корреляции** ::
  - Для двух детерминированных сигналов конечной энергии: наложение одного на другой с временным сдвигом → измерение «перекрытия» → число.
  - При изменении сдвига получается функция → это корреляционная функция.

- **Физический смысл корреляционной функции** ::
  - Показывает, насколько один сигнал «совпадает» с другим при данном временном сдвиге.

## Основные термины, описывающие случайную величину

- **Распределение случайной величины** (распределение вероятностей) ::
  - Правило, описывающее, как вероятность распределена по возможным значениям случайной величины.
  - Показывает, как вероятность «размазана» по числовой оси (где она сосредоточена).
  - Аналогия из физики: плотность массы.
  - Способы задания:
    - функция распределения
    - плотность вероятности (для непрерывных СВ)
    - закон / ряд распределения (для дискретных СВ)

- **Взвешивание** ::
  - Операция присвоения разным элементам набора данных различных весов (коэффициентов значимости).

## Классификация случайных величин

- **Дискретные случайные величины** ::
- **Непрерывные случайные величины** ::

### Дискретная случайная величина

- **Определение** ::
  - Случайная величина, принимающая конечное (или счётное) количество значений.

- **Чем полностью определяется** ::
  - Законом распределения (рядом распределения: значение ↔ вероятность).

### Непрерывная случайная величина

- **Определение** ::
  - Случайная величина, значения которой принадлежат некоторому интервалу (или всей числовой прямой).

- **Чем полностью определяется** ::
  - Функцией плотности вероятности.

## Числовые характеристики случайной величины

### Математическое ожидание (МО)

- **Определение** ::
  - Среднее значение случайной величины, взвешенное по её распределению.

- **Физический смысл** ::
  - Значение, к которому стремится среднее арифметическое при многократном повторении эксперимента и усреднении результатов.

- **Сравнение с дисперсией** ::
  - МО — центр распределения (положение).
  - Дисперсия — ширина / разброс относительно этого центра.

- **Пример из радиотехники** ::
  - МО теплового шума на выходе РПрУ в нормальных условиях равно 0 (симметричные колебания относительно нуля).

### Дисперсия

- **Определение** ::
  - Мера разброса значений случайной величины относительно математического ожидания.

- **Формула** ::
  - $$ D(X) = M(X^2) - [M(X)]^2 = \sigma^2 $$
  - где $\sigma^2$ — дисперсия, $\sigma$ — среднеквадратическое отклонение (СКО).

- **Альтернативная трактовка формулы** ::
  - Математическое ожидание квадрата отклонения от среднего.

- **Почему квадрат?** ::
  - Чтобы отклонения в обе стороны (+ и −) не компенсировали друг друга, а всегда вносили положительный вклад.

## Функция плотности вероятности (ПРВ, pdf)

- **Формальное определение** ::
  - Производная функции распределения (когда она дифференцируема).

- **Формула** ::
  - $$ f(x) = \frac{dF(x)}{dx} $$
  - где $F(x)$ — функция распределения, $x$ — значение случайной величины.

- **Что показывает** ::
  - Интенсивность накопления вероятности в окрестности точки $x$ (локальная концентрация вероятности).

- **Является ли сама вероятностью?** ::
  - Нет — это плотность вероятности (размерность: 1 / [единица измерения СВ]).

- **Для каких величин существует** ::
  - Только для непрерывных случайных величин.
