Они будут абсолютно одинаковыми. Более подробно: [[id:f7169f46-db6e-4ad0-bbe4-fc6e7272e567][математическое доказательство периодичности спектра дискретного сигнала]].[fc:0]

- Уточнение по поводу периода повоторения спектра дискретного сигнала :: Синусоиды с частотами, отличающимися на \(kf_s\), где \(k\) -- целое число, дают *идентичные последовательности отсчетов*. Это означает, что сигналы с частотами \(f, f + f_s, f + 2f_s, f - f_s, f - 5f_s\) и т. д. *неотличимы друг от друга* после дискретизации.[fc:2]
- Следствие периодичности спектра дискретного сигнала ::
  - Эффект [[id:b715af01-9c77-46ec-a77f-6a580b5a162a][алиасинга]], при котором высокие частоты могут "маскироваться" под более низкие.
  - [[id:7d06be95-a0be-4be5-99f1-a6a1e9684dee][Условие однозначного восстановления непрерывного сигнала, имеющего ограниченный по ширине спектр по его дискретным отсчетам]], которое призвано устранить возникновение эффекта алиасинга.[fc:3]

** Частота дискретизации
- #π/определение:
  - *Частота дискретизации* -- частота взятия отсчётов сигнала при его дискретизации.
- #π/формула:
  - \(f_{\text{д}}=\dfrac{1}{\Delta t}\).
- #π/условие однозначного восстановления сигнала:
  - Для однозначного восстановления сигнала /частота дискретизации/ должна более чем в два раза превышать наибольшую частоту в спектре сигнала.
  - #π/формула для данного условия:
    - \(f_{\text{д}}\geq f_{M}\).
