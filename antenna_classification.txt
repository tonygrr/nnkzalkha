# Кратко к изучению
## Коэффициент усиления антенны
- Фундоментальное определение (полностью соответствует формуле КУ) :: Это отношение [[id:b0c0a96e-8ae8-48de-a3a8-1b3904199b66][интенсивности излучения]] антенны в заданном направлении к интенсивности излучения изотропного излучателя, к которому подведена та же входная мощность $P_{in}\).
- Определение через КНД :: Это [[id:5721d995-2c4a-4fea-acfe-5f792f28076f][коэффициент направленного действия]] антенны с учетом всех потерь
- Принципиальное различие КУ и КНД :: В отличие от КНД, КУ учитывает все потери в антенне: омические потери, потери на рассогласование и т. д.
- Физический смысл :: Показывает, во сколько раз антенна создаёт большую интенсивность излучения в направлении максимума по сравнению с изотропным излучателем при той же подводимой мощности. Это главный практический параметр антенны: именно КУ используется в расчётах энергетики радиолинии.
- Обозначение :: $G$ -- в отечественной и зарубежной литературе; реже $K$ -- в старой отечественной литературе.
- Единицы измерения :: дБи (относительно изотропного излучателя), дБд (относительно полуволнового диполя; значение на 2.15 дБ меньше, чем в дБи).

### Формулы
- Точная формула :: $G(\theta,\varphi)=\dfrac{4\pi \cdot U(\theta,\varphi)}{P_{in}}\), где
  - $U(\theta,\varphi)$ -- интенсивность излучения в направлении $(\theta,\varphi)\), Вт/ср;
  - $P_{in}$ -- мощность, подведённая ко входу антенны, Вт.
- Формула через КНД :: $G=\eta \cdot D\), где
  - $\eta$ -- полный КПД антенны, $0 \leq \eta \leq 1\);
  - $D$ -- коэффициент направленного действия.
- Полный КПД :: $\eta = \eta_{ref} \cdot \eta_{ohm}\), где
  - $\eta_{ref} = 1 - |\Gamma|^{2}$ -- КПД по рассогласованию;
  - $\Gamma$ -- коэффициент отражения на входе антенны;
  - $\eta_{ohm}$ -- КПД по омическим потерям.
- Формула через нормированную ДН :: $G_0=\dfrac{\eta \cdot 4\pi}{\displaystyle\int_0^{\pi}\int_0^{2\pi} F_0(\theta,\varphi)\sin\theta \, d\theta \, d\varphi}\)
  - Следствие :: Аналогична формуле КНД с множителем $\eta\).
    Форма ДН определяет КНД, КПД масштабирует его до КУ.
- Сравнение с КНД :: $G \leq D$ всегда. Равенство достигается
  только у идеальной антенны без потерь (\(\eta = 1\)).

### Методы измерения КУ на практике
#### Метод двух антенн
- Суть :: Два одинаковых экземпляра исследуемой антенны располагают друг напротив друга и измеряют отношение принятой мощности к переданной.
- Формула :: $\dfrac{P_r}{P_t} = G^{2}\left(\dfrac{\lambda}{4\pi R}\right)^{2}\), где
  - $P_r$ -- принятая мощность, Вт;
  - $P_t$ -- переданная мощность, Вт;
  - $R$ -- расстояние между антеннами, м;
  - $\lambda$ -- длина волны, м.
- Ограничение :: Требует двух идентичных экземпляров антенны.

#### Метод трёх антенн
- Суть :: Используют три разные антенны, проводят три
  попарных измерения. Получают систему из трёх уравнений
  и решают её относительно $G_1, G_2, G_3\).
- Преимущество :: Не требует одинаковых антенн.

#### Метод эталонной антенны (метод замещения)
- Суть :: Наиболее распространён при поверке. КУ исследуемой антенны определяется относительно эталонной антенны с известным КУ (обычно стандартный рупор).
- Формула :: $G_{DUT} = G_{ref} + \Delta P_{dB}\), где
  - $G_{ref}$ -- КУ эталонной антенны, дБи;
  - $\Delta P_{dB}$ -- разность принятых мощностей, дБ.

#### Метод сканера ближнего поля
- Суть :: Измеряется трёхмерная ДН через NF-FF преобразование, затем КУ вычисляется численно по точной формуле с учётом КПД антенны.

### Связь КУ с коэффициентом калибровки антенны
- Коэффициент калибровки (Antenna Factor) :: $AF = E / U\), где - $E$ -- напряжённость поля в точке установки антенны, В/м; - $U$ -- напряжение на выходе антенны, В.
- Физический смысл :: Чем выше КУ антенны, тем эффективнее она принимает поле, тем меньше $AF\).
- Формула пересчёта (для согласованной антенны, $Z_0 = 50$ Ом) :: $AF_{dB} = 20\lg f_{MHz} - G_{dBi} - 29{,}8\)
- Применение :: $AF$ указывается в калибровочном свидетельстве измерительной антенны и используется для пересчёта показаний приёмника в значения напряжённости поля при измерениях ЭМС.

## Классификация антенн по форме ДН
### Тороидальная ДН
- Форма :: Максимум в плоскости, перпендикулярной оси антенны; провал вдоль оси. Напоминает форму бублика (тора).
- Симметрия :: Азимутально симметрична --- ДН одинакова по всем горизонтальным направлениям.
- Антенны :: Штыревые, дипольные (полуволновый и вибраторный), петлевые, щелевые (горизонтальная щель).
- Для оценки КНД :: Применяются формулы Макдональда и Позара --- достаточно одного угла $\Delta\theta_d$ в угломестной плоскости.

### Направленная ДН (карандашная)
- Форма :: Один узкий главный лепесток в заданном направлении, боковые лепестки значительно слабее.
- Антенны :: Рупорные, параболические, апертурные, патч-антенны (микрополосковые), антенные решётки,
  логопериодические (широкий карандашный лепесток вдоль оси).
- Для оценки КНД :: Применяются формулы Краусса, Эллиота, Тей и Перейра --- требуются два угла $\theta_{1d}$ и $\theta_{2d}\).

- Антенны :: Антенны с вытянутой прямоугольной апертурой, цилиндрические параболические антенны.

### Косекансная ДН
- Форма :: Специально сформированная несимметричная ДН: уровень излучения растёт с углом по закону $1/\sin\theta\).
- Назначение :: Обеспечивает равномерное покрытие земной поверхности при обзорной РЛС --- компенсирует убывание отражённого сигнала с расстоянием.
- Антенны :: Апертурные антенны и антенные решётки РЛС кругового и секторного обзора.

### Многолучевая ДН
- Форма :: Несколько одновременных узких лучей в разных направлениях.
- Антенны :: Антенные решётки с цифровым диаграммообразованием (ЦДО), линзовые антенны (матрица Батлера).
  
## Классификация антенн
### По принципу излучения
- Антенны бегущей волны :: Излучение возникает вследствие распространения бегущей волны вдоль проводника или апертуры. Примеры: логопериодические, спиральные, щелевые антенны.
- Антенны стоячей волны :: Излучение возникает вследствие резонанса. Примеры: дипольные, штыревые, рамочные антенны.
- Апертурные антенны :: Излучение происходит через раскрыв (апертуру) заданной формы. Примеры: рупорные, параболические, щелевые антенны.
- Антенные решётки :: Излучение формируется суперпозицией полей от множества элементарных излучателей.

### По форме ДН
- Тороидальная ДН :: Штыревые, дипольные, петлевые антенны.
- Игольчатая ДН (pencil beam) :: Рупорные, параболические, апертурные антенны, антенные решётки.
- Веерная ДН :: Антенны с вытянутой прямоугольной апертурой, цилиндрические параболические антенны.
- Косекансная ДН :: Апертурные антенны и решётки РЛС обзора.
- Многолучевая ДН :: Антенные решётки с ЦДО, линзовые антенны.

### По диапазону частот
- КВ (3--30 МГц) :: Вибраторные, ромбические, рамочные антенны.

### Основные типы антенн
#### Дипольная антенна
- Конструкция :: Два симметричных проводника, питаемых в центре. Полуволновый диполь --- наиболее распространённый вариант с длиной каждого плеча $\lambda/4\).
- Форма ДН :: Тороидальная. Максимум излучения перпендикулярен оси диполя, нуль --- вдоль оси.
- КУ :: Полуволновый диполь: 2.15 дБи. Используется как эталон (0 дБд).
- Импеданс :: ~73 Ом для полуволнового диполя в свободном пространстве.
- Поляризация :: Линейная, вдоль оси диполя.
- Диапазон применения :: КВ, УКВ, ОВЧ. Широко применяется как элемент антенных решёток.
- Особенности :: Простейшая резонансная антенна. КУ полуволнового диполя является опорным значением --- шкала дБд. Используется как эталонная антенна при измерениях и как элемент сравнения в определении КНД.
- Применение :: Широковещательные передатчики, базовые станции, элементы решёток, эталонные антенны в измерительных системах.

#### Штыревая антенна (монополь)
- Конструкция :: Вертикальный проводник длиной $\lambda/4$ над проводящей плоскостью (земля или противовесы). Фактически --- половина диполя с зеркальным изображением в плоскости.
- Форма ДН :: Половина тороидальной --- максимум в горизонтальной плоскости, нуль вдоль оси. В пространстве над плоскостью земли.
- КУ :: ~5.15 дБи (на 3 дБ выше диполя за счёт излучения только в полупространство).
- Импеданс :: ~36.5 Ом (половина от импеданса диполя).
- Поляризация :: Вертикальная линейная.
- Диапазон применения :: КВ, УКВ. Автомобильные антенны, базовые станции, корабельные и самолётные антенны.
- Особенности :: Требует хорошо проводящей плоскости или системы противовесов. Качество «земли» существенно влияет на КУ и форму ДН.
- Применение :: Мобильная связь, морская связь, AM-вещание.

#### Рамочная (петлевая) антенна
- Конструкция :: Замкнутый проводник в форме прямоугольника, квадрата или круга. Малая рамка --- периметр много меньше $\lambda\). Большая рамка --- периметр порядка $\lambda\).
- Форма ДН :: Малая рамка: тороидальная, ось тора перпендикулярна плоскости рамки (двойная восьмёрка в плоскости рамки). Большая рамка: максимум перпендикулярен плоскости рамки.
- КУ :: Малая рамка: ~1.5 дБи. Большая рамка: ~3--4 дБи.
- Поляризация :: Линейная, перпендикулярна плоскости рамки.
- Особенности :: Малая рамка нечувствительна к полю вдоль своей оси --- это используется для пеленгации. В сочетании со штыревой антенной даёт кардиоидную ДН (пеленгатор с однозначным определением направления).
- Применение :: Пеленгаторы, КВ-приёмники, RFID, антенны для измерений магнитного поля при ЭМС-тестировании.

#### Антенна Яги--Уда
- Конструкция :: Линейная решётка из параллельных диполей: один активный вибратор, один рефлектор (чуть длиннее $\lambda/2\)) сзади, несколько директоров (чуть короче $\lambda/2\)) спереди.
- Форма ДН :: Игольчатая, направление максимума --- вдоль оси решётки, в сторону директоров.
- КУ :: 6--20 дБи в зависимости от числа элементов.
- Поляризация :: Линейная, вдоль плоскости элементов.
- Диапазон применения :: УКВ, ОВЧ (30--1000 МГц).
- Особенности :: Полоса согласования узкая (5--10%). Простая конструкция при относительно высоком КУ. КУ растёт с числом директоров, но нелинейно --- прирост уменьшается с добавлением каждого нового элемента.
- Применение :: Телевизионные антенны, радиолюбительская связь, WiFi, направленные линии связи.

#### Логопериодическая антенна (LPDA)
- Конструкция :: Система диполей с экспоненциально изменяющимися длинами и расстояниями. Смежные диполи подключаются с инверсией фазы. Геометрический масштабный коэффициент: $\tau = l_{n+1}/l_n = d_{n+1}/d_n\).
- Форма ДН :: Игольчатая (широкий карандашный лепесток) вдоль оси в сторону более коротких элементов.
- КУ :: 6--12 дБи (ниже, чем у Яги той же физической длины).
- Поляризация :: Линейная.
- Диапазон применения :: УКВ, СВЧ. Рабочая полоса перекрывает декаду и более (например, 200 МГц -- 2 ГГц).
- Особенности :: Главное достоинство --- широкая полоса рабочих частот при относительно постоянных характеристиках (КУ, импеданс, ДН). В каждый момент времени активны лишь несколько элементов, образующих «активную зону» вблизи резонансного диполя.
- Применение :: Широкополосные измериПри этом можешь пользоваться файлом, который я прикреплю. Можешь на нем основываться.тельные антенны при ЭМС-тестировании (один из основных рабочих инструментов в диапазоне 200 МГц -- 2 ГГц), телевизионные антенны, разведывательные приёмники.

#### Рупорная антенна
- Конструкция :: Расширяющийся металлический волновод прямоугольного или круглого сечения. Основные виды: секториальный рупор (расширяется в одной плоскости), пирамидальный (расширяется в обеих плоскостях), конический (круглое сечение).
- Форма ДН :: Игольчатая. Ширина луча определяется размером раскрыва в длинах волн: $\Delta\theta \approx \lambda / L\).
- КУ :: 10--25 дБи в зависимости от размера раскрыва. Стандартные измерительные рупоры: 15--25 дБи.
- Поляризация :: Линейная (прямоугольный рупор), круговая или линейная (конический рупор с соответствующим возбуждением).
- УБЛ :: -20 дБ и ниже относительно главного лепестка.
- Диапазон применения :: СВЧ, КВЧ (1--100 ГГц и выше).
- Особенности :: Хорошо предсказуемые характеристики, которые поддаются точному расчёту. Умеренная полоса частот (30--50%). Оптимальный рупор --- компромисс между размером апертуры и фазовыми искажениями поля в раскрыве.
- Применение :: Основная эталонная антенна при калибровке и поверке в СВЧ-диапазоне. Облучатели параболических антенн. ЭМС-измерения в диапазоне 1--18 ГГц. Питающие антенны в безэховых камерах.
- Метрологическая роль :: Стандартные измерительные рупоры (Standard Gain Horn, SGH) имеют паспортные значения КУ с погрешностью ±0.3--0.5 дБ. Используются как эталоны второго разряда при поверке методом замещения.

#### Параболическая антенна
- Конструкция :: Параболический рефлектор (зеркало) с облучателем в фокусе. Облучатель --- обычно рупор или диполь с рефлектором. Параболоид фокусирует падающую плоскую волну в точку (и наоборот).
- Форма ДН :: Игольчатая. Ширина луча: $\Delta\theta \approx 70\lambda / D$ (в градусах), где $D$ --- диаметр зеркала.
- КУ :: $G = \eta_a \cdot ({\pi D}/{\lambda})^2\), где $\eta_a$ --- КПД апертуры (обычно 0.55--0.65). Типичные значения: 30--55 дБи.
- Поляризация :: Определяется облучателем; возможна любая.
- УБЛ :: -20...-30 дБ и ниже при хорошем профиле подсвета зеркала.
- Диапазон применения :: СВЧ, КВЧ (1--100 ГГц).
- Особенности :: Наивысший КУ среди одноапертурных антенн при заданном диаметре. Узкая полоса согласования облучателя ограничивает рабочую полосу. Блокировка апертуры облучателем снижает КУ и ухудшает УБЛ --- решается офсетной схемой (облучатель вынесен за пределы раскрыва).
- Применение :: Спутниковая связь и навигация, радиотелескопы, РЛС, радиорелейные линии, глубокий космос (антенны DSN до 70 м в диаметре).

#### Патч-антенна (микрополосковая антенна)
- Конструкция :: Металлическая пластина (патч) над проводящей плоскостью, разделённые диэлектрической подложкой. Питание --- через коаксиальный зонд или микрополосковую линию. Форма патча: прямоугольная, круглая, кольцевая и др.
- Форма ДН :: Широкая игольчатая, перпендикулярно плоскости патча. Ширина луча порядка 60--100°.
- КУ :: 5--9 дБи. Невысокий по сравнению с рупором или параболой.
- Поляризация :: Линейная или круговая (при специальной форме патча или двойном питании).
- Полоса :: Узкополосна (1--5%) для одиночного патча. Расширяется до 20--30% при использовании многослойных структур и паразитных элементов.
- Диапазон применения :: СВЧ (0.5--60 ГГц).
- Особенности :: Предельно малый профиль и масса --- возможна интеграция непосредственно в корпус устройства. Легко масштабируется в решётки. Низкий КПД из-за потерь в диэлектрике подложки.
- Применение :: GPS/ГЛОНАСС-приёмники, мобильные телефоны, RFID, бортовые антенны беспилотников, базовые станции 5G (решётки патчей).

#### Спиральная антенна
- Конструкция :: Проводник, намотанный по цилиндрической или конической спирали. Два режима работы:
  - Нормальный режим (диаметр $\ll \lambda\)): тороидальная ДН, поляризация линейная;
  - Осевой режим (длина витка $\approx \lambda\)): игольчатая ДН вдоль оси,
    поляризация круговая.
- КУ :: В осевом режиме: 10--20 дБи.
- Диапазон применения :: УКВ, СВЧ.
- Особенности :: Осевой режим обеспечивает широкую полосу (2:1 и более) при стабильной круговой поляризации. Это уникальное сочетание --- мало других антенн обеспечивают одновременно широкую полосу, направленность и круговую поляризацию.
- Применение :: Спутниковая связь, космические аппараты, GPS, системы телеметрии ракет.

#### Щелевая антенна
- Конструкция :: Щель в проводящем экране (металлическом листе или стенке волновода). По принципу двойственности Бабине --- дуальна дипольной антенне: поля взаимно переставлены.
- Форма ДН :: Вертикальная щель: тороидальная (горизонтальная поляризация). Горизонтальная щель: тороидальная (вертикальная поляризация). Щели в волноводе при надлежащем расположении --- игольчатая ДН.
- КУ :: Одиночная щель: ~2 дБи. Решётка щелей: 15--25 дБи.
- Поляризация :: Перпендикулярна длине щели (двойственность с диполем).
- Диапазон применения :: СВЧ, КВЧ.
- Особенности :: Конструктивно является частью несущей поверхности --- удобна для интеграции в корпус воздушного судна или корабля. Решётки щелей в волноводе обеспечивают высокий КУ при малом профиле.
- Применение :: Бортовые РЛС, антенны самолётов и кораблей, антенны базовых станций, измерительные антенны.

#### Вибраторные антенны (общий класс)
- Турникетная антенна :: Два ортогональных диполя с питанием со сдвигом 90° --- круговая поляризация в горизонтальной плоскости. Применяется в телевещании (ОВЧ).
- Коллинеарная антенна :: Несколько диполей, расположенных вдоль одной вертикальной оси и синфазно возбуждаемых. Тороидальная ДН, но более узкая по углу места, чем у одиночного диполя. КУ: 5--10 дБи. Применяется в базовых станциях и вещании.
- Ромбическая антенна :: Горизонтальный ромб из проводников, согласованный на дальнем конце. Широкополосная (3:1 и более), игольчатая ДН под углом к плоскости антенны. КУ: 10--15 дБи. КВ-диапазон, дальняя связь.

#### Антенная решётка
- Конструкция :: Совокупность излучающих элементов с управляемыми амплитудой и фазой возбуждения каждого элемента.
- Форма ДН :: Определяется множителем решётки. При надлежащем законе возбуждения --- игольчатая или веерная.
- КУ :: Пропорционален числу элементов $N\): $G \approx N \cdot G_{el}\), где $G_{el}$ --- КУ одного элемента. Для больших решёток: 30--50 дБи.
- Особенности :: Позволяет управлять формой и направлением ДН электронными методами без механического поворота. Возможно формирование нескольких лучей одновременно.
  - Влияние числа элементов $N$ :: Главный лепесток сужается, КУ растёт пропорционально $N\). УБЛ не меняется при равноамплитудном возбуждении (остаётся -13.5 дБ).
  - Влияние шага решётки $d$ :: При $d > \lambda/2$ появляются паразитные дифракционные максимумы. Условие их отсутствия: $d \leq \lambda/2\).
  - Управление лучом :: Линейный фазовый прогресс $\psi$ между элементами отклоняет луч на угол $\theta_0 = \arcsin(\psi / kd)\).
  - Управление УБЛ :: Применение весовых окон (Тейлор, Чебышёв) снижает УБЛ за счёт уширения главного лепестка.
- Типы решёток ::
  - АФАР (Активная ФАР) :: Каждый элемент имеет собственный приёмо-передающий модуль. Наивысшая гибкость управления ДН, высокая надёжность.
  - ПФАР (Пассивная ФАР) :: Один мощный передатчик, фаза каждого элемента управляется фазовращателем.
  - Цифровые решётки (ЦДО) :: Аналого-цифровое преобразование на уровне элемента или подрешётки; формирование ДН в цифровой области.
- Применение :: РЛС (обзорные, наведения, противоракетной обороны), системы связи 5G/6G, радиоастрономия, радиоэлектронная борьба, спутниковые группировки.

### Применение антенн в измерениях и метрологии
#### Эталонные и измерительные антенны
- Стандартный рупор (Standard Gain Horn, SGH) ::
  - Диапазон :: 1--40 ГГц (разные типоразмеры).
  - КУ :: 15--25 дБи с погрешностью ±0.3--0.5 дБ.
  - Роль :: Эталон второго разряда при поверке методом замещения. Облучатель в безэховых камерах и на антенных полигонах.
- Полуволновый диполь ::
  - Диапазон :: 30 МГц -- 1 ГГц.
  - Роль :: Эталонная антенна при ЭМС-измерениях (CISPR, EN 55016). Единица шкалы дБд.
- Логопериодическая антенна (LPDA) ::
  - Диапазон :: 200 МГц -- 2 ГГц (один типоразмер).
  - Роль :: Основная широкополосная измерительная антенна при ЭМС-тестировании. Применяется при измерениях индустриальных помех по ГОСТ Р, CISPR 22/32.
- Биконическая антенна ::
  - Конструкция :: Два конуса, питаемых в вершинах. Широкополосный вариант диполя.
  - Диапазон :: 20 МГц -- 300 МГц.
  - Роль :: Широкополосные ЭМС-измерения в диапазоне 20--300 МГц.

#### Применение антенн в системах сканирования ближнего поля
- Зондовые антенны :: Открытый конец прямоугольного волновода, малый рупор или диполь. Сканируют поверхность и измеряют амплитуду и фазу ближнего поля для последующего NF-FF преобразования.
- Требования к зонду ::
  - Малый размер для высокого пространственного разрешения.
  - Известная ДН (учитывается в алгоритме пересчёта NF-FF как поправка).
  - Стабильные характеристики во всём диапазоне сканирования.

# Характеристики антенн и антенные измерения

## Прочие важные характеристики антенны

### Входной импеданс

Входной импеданс $Z_{in} = R_{in} + jX_{in}$ — комплексное сопротивление антенны на входных клеммах. Активная часть $R_{in}$ включает сопротивление излучения $R_{rad}$ (мощность, ушедшая в эфир) и сопротивление потерь $R_{loss}$ (мощность, рассеянная в виде тепла). Реактивная часть $X_{in}$ определяет расстройку антенны от резонанса.

КПД антенны напрямую связан с импедансом:

$$\eta_{ohm} = \frac{R_{rad}}{R_{rad} + R_{loss}}$$

На практике входной импеданс измеряется векторным анализатором цепей (ВАЦ) и является исходной характеристикой для всех дальнейших расчётов согласования.

### КСВ и коэффициент отражения

Коэффициент стоячей волны (КСВ) и коэффициент отражения $\Gamma$ — характеристики согласования антенны с питающим трактом (обычно 50 Ом). Связаны с импедансом:

$$\Gamma = \frac{Z_{in} - Z_0}{Z_{in} + Z_0}, \qquad \text{КСВ} = \frac{1 + |\Gamma|}{1 - |\Gamma|}$$

Физически: КСВ = 1 означает полное согласование, вся мощность передаётся в антенну. КСВ = 2 означает, что ~11% мощности отражается обратно в генератор. КСВ = ∞ — полное рассогласование (разомкнутая или короткозамкнутая нагрузка).

На практике нормируют КСВ ≤ 1.5 (потери на отражение не более 4%) или ≤ 2. Параметр $S_{11}$ в логарифмическом виде — это «возвратные потери» (Return Loss):

$$RL_{dB} = -20\lg|\Gamma|$$

При $\Gamma = 0$ — $RL = \infty$ дБ (идеальное согласование). При КСВ = 2 — $RL \approx 9.5$ дБ.

### Рабочая полоса частот

Полоса частот — диапазон, в котором характеристики антенны (КУ, КСВ, ДН, поляризация) укладываются в заданные допуски. Определяется по разным критериям в зависимости от задачи:

- **По КСВ:** полоса, в которой КСВ ≤ 2 (наиболее распространённый критерий)
- **По КУ:** полоса, в которой КУ снижается не более чем на 3 дБ от максимума
- **По ДН:** полоса, в которой ширина ДН не выходит за заданные пределы

Разные типы антенн кардинально отличаются по полосе: вибраторные резонансные — единицы процентов, логопериодические — декады, спиральные в осевом режиме — октавы.

### Поляризационные характеристики

Поляризация — ориентация вектора электрического поля в пространстве. Различают:

- **Линейная поляризация** — вектор $\vec{E}$ колеблется в одной плоскости. Горизонтальная или вертикальная — зависит от ориентации антенны.
- **Круговая поляризация** — вектор $\vec{E}$ вращается с постоянной амплитудой. Правая (RHCP) или левая (LHCP). Формируется двумя ортогональными компонентами с равными амплитудами и сдвигом фаз 90°.
- **Эллиптическая поляризация** — общий случай: вектор $\vec{E}$ описывает эллипс. Линейная и круговая — частные случаи.

Основные параметры поляризации:

**Осевое отношение (Axial Ratio, AR)** — отношение большой полуоси поляризационного эллипса к малой. AR = 1 (0 дБ) — идеальная круговая поляризация. AR → ∞ — линейная поляризация. Норма для «хорошей» круговой поляризации: AR ≤ 3 дБ.

**Поляризационная развязка (Cross-Polarization Discrimination, XPD)** — отношение уровня излучения в основной поляризации к уровню в кросс-поляризации. Измеряется в дБ. Для качественных антенн XPD ≥ 20–30 дБ в направлении главного лепестка.

**Поляризационные потери** возникают при несовпадении поляризации передающей и приёмной антенн. При ортогональных поляризациях (горизонтальная + вертикальная, RHCP + LHCP) — потери теоретически бесконечны, сигнал не принимается.

### Эффективная площадь апертуры

Эффективная площадь апертуры $A_{eff}$ — характеристика приёмной антенны, показывающая, какую площадь сечения электромагнитной волны антенна «перехватывает»:

$$P_{rec} = S \cdot A_{eff}$$

где $S$ — плотность потока мощности падающей волны (Вт/м²), $P_{rec}$ — принятая мощность.

Связь с КУ фундаментальна:

$$A_{eff} = \frac{\lambda^2}{4\pi} \cdot G$$

Это соотношение универсально — оно справедливо для любой антенны, независимо от её конструкции. Из него следует парадоксальный на первый взгляд факт: изотропный излучатель имеет ненулевую эффективную площадь $A_{eff} = \lambda^2 / 4\pi$. Физически — это минимальная площадь, которую способна иметь антенна в данной точке пространства.

Эффективная площадь входит в формулу Фриса для расчёта мощности в радиолинии:

$$\frac{P_r}{P_t} = G_t \cdot G_r \cdot \left(\frac{\lambda}{4\pi R}\right)^2$$

### Фазовый центр антенны

Фазовый центр — точка в пространстве, из которой излучаемое поле выглядит как сферическая волна с постоянной фазой в пределах главного лепестка. Для идеального точечного излучателя фазовый центр совпадает с его физическим положением.

Фазовый центр критически важен при использовании антенны в качестве облучателя параболического зеркала: он должен точно совпадать с фокусом параболы, иначе фазовый фронт в раскрыве будет искажён и КУ упадёт. Также фазовый центр важен в навигационных антеннах GPS/ГЛОНАСС — его нестабильность вносит ошибку в измерение фазы несущей.

### Время группового запаздывания

Group Delay — производная фазы по угловой частоте: $\tau = -d\phi/d\omega$. Постоянное групповое запаздывание означает, что все частотные составляющие сигнала проходят через антенну с одинаковой задержкой — фазовые соотношения сохраняются и импульс не искажается. Неравномерное групповое запаздывание приводит к дисперсии: импульс «размывается». Критично для широкополосных систем — сверхширокополосных (UWB) антенн, радаров с зондирующим импульсом, а также для прецизионной навигации.

## Диаграмма Вольперта–Смита

### Что это такое

Диаграмма Смита (в отечественной литературе — диаграмма Вольперта–Смита, по именам советского учёного Л. А. Вольперта и американского инженера Ф. Смита, независимо разработавших её в 1930-х) — это графическое представление комплексного коэффициента отражения $\Gamma$ в полярных координатах, на которое нанесены линии постоянного нормированного сопротивления и реактанса.

### Построение

Нормированный импеданс: $z = Z_{in}/Z_0 = r + jx$, где $Z_0$ — характеристическое сопротивление (обычно 50 Ом).

Диаграмма — это единичная окружность в плоскости $\Gamma$. Внутри неё нанесены:

- **Окружности постоянного $r$** — семейство окружностей, касающихся правой точки (1,0). При $r = 0$ — вся левая граница окружности. При $r = 1$ — окружность, проходящая через центр. При $r → ∞$ — точка (1,0).
- **Дуги постоянного $x$** — семейство дуг, проходящих через точку (1,0). Верхняя половина — индуктивный реактанс ($x > 0$), нижняя — ёмкостный ($x < 0$).

Ключевые точки диаграммы:

- **Центр** (0,0) — $Z_{in} = Z_0$, идеальное согласование, $\Gamma = 0$, КСВ = 1
- **Левая точка** (-1,0) — $Z_{in} = 0$, короткое замыкание
- **Правая точка** (1,0) — $Z_{in} = ∞$, холостой ход
- **Окружности КСВ** — концентрические окружности вокруг центра; радиус = $|\Gamma|$

### Как пользоваться

1. **Читать импеданс.** Точка на диаграмме однозначно задаёт $Z_{in}$. Находишь пересечение нужных окружностей $r$ и дуг $x$ — читаешь нормированный импеданс, умножаешь на $Z_0$.

2. **Оценивать КСВ.** Радиус окружности через данную точку (от центра) — это $|\Gamma|$. КСВ читается со вспомогательной шкалы или вычисляется по формуле.

3. **Двигаться по линии передачи.** При движении вдоль линии без потерь точка движется по окружности постоянного $|\Gamma|$ (постоянного КСВ) — по часовой стрелке при движении от нагрузки к генератору. Полный оборот соответствует $\lambda/2$ длины линии. Это позволяет находить импеданс в любом сечении линии.

4. **Проектировать согласование.** Задача согласования — переместить точку из произвольного положения в центр диаграммы. Шунтирующая ёмкость движет точку по окружности постоянной проводимости (удобнее работать с допускной диаграммой — инвертированной). Последовательная индуктивность — по дуге постоянного $r$. Четвертьволновый трансформатор — поворот на 180° по окружности КСВ.

5. **Анализировать АЧХ импеданса.** При сканировании по частоте точка движется по диаграмме, рисуя траекторию. Петля вокруг центра — резонансная антенна. Насколько близко к центру проходит траектория — насколько антенна согласована в полосе.

На практике диаграмма Смита является стандартным интерфейсом отображения результатов в ВАЦ (VNA). Все современные приборы (Keysight, Rohde & Schwarz, Copper Mountain) отображают $S_{11}$ в виде диаграммы Смита наряду с прямоугольными форматами.

## Основные антенные измерения

### Измерение КСВ и входного импеданса

**Метод:** Векторный анализатор цепей (ВАЦ, VNA). Измеряется $S_{11}$ — нормированный коэффициент отражения в логарифмическом (дБ) или комплексном виде.

**Процедура:**
1. Калибровка ВАЦ в опорной плоскости (SOL — Short/Open/Load или более точные методы SOLT, TRL)
2. Подключение антенны к измерительному порту
3. Сканирование по частоте; результат отображается как $S_{11}(f)$, КСВ$(f)$ или на диаграмме Смита

**Что получают:** Рабочую полосу частот, резонансные частоты, импеданс, КПД по рассогласованию.

**Требования к условиям:** Измерение желательно проводить в условиях, близких к реальным (вдали от металлических предметов), либо в безэховой камере — окружающие предметы влияют на импеданс антенны через взаимные связи.

### Измерение диаграммы направленности

**Метод 1 — Дальняя зона (Far-Field Range):**
Антенна-под-испытанием (AUT) устанавливается на поворотном устройстве. На фиксированном расстоянии (дальняя зона: $R \geq 2D^2/\lambda$) размещается опорная антенна. AUT поворачивается; принимаемый или излучаемый сигнал записывается как функция угла. Строится ДН.

Недостаток: при большом размере антенны требуемое расстояние дальней зоны может составлять сотни метров.

**Метод 2 — Компактный антенный полигон (Compact Antenna Test Range, CATR):**
Коллимирующее зеркало или линза формирует плоский фазовый фронт уже на расстоянии нескольких метров от источника. AUT облучается «искусственной» плоской волной. Позволяет измерять крупногабаритные антенны в относительно небольшой камере.

**Метод 3 — Сканер ближнего поля:**
Как описывалось ранее: измерение ближнего поля по поверхности (плоской, цилиндрической, сферической) с последующим NF-FF преобразованием. Сферический сканер — наиболее полный метод, даёт полную трёхмерную ДН.

**Что получают:** Форму ДН, ширину главного лепестка, УБЛ, уровень задних лепестков, положение нулей.

### Измерение коэффициента усиления

**Метод абсолютного КУ (метод трёх антенн):**
Три попарных измерения коэффициента передачи $S_{21}$ между антеннами. Система трёх уравнений Фриса решается относительно $G_1$, $G_2$, $G_3$. Не требует эталонов, результат — абсолютный КУ.

**Метод замещения (Gain Transfer / Comparison Method):**
Наиболее распространён на практике. Вместо AUT поочерёдно подключается эталонная антенна (стандартный рупор SGH с паспортным КУ). Разность уровней $S_{21}$ в дБ даёт разность КУ. Точность определяется точностью КУ эталона — обычно ±0.3–0.5 дБ.

**Метод через ближнее поле:**
После NF-FF преобразования КУ вычисляется численно из трёхмерной ДН с учётом КПД (последний определяется отдельно из $S_{11}$ и тепловых потерь).

**Важный нюанс:** все методы измерения КУ требуют поляризационного совмещения AUT и опорной антенны — поляризации должны совпадать. Иначе результат будет занижен на величину поляризационных потерь.

### Измерение поляризационных характеристик

**Метод поляризационной диаграммы:**
Опорная антенна с линейной поляризацией вращается вокруг оси, направленной на AUT. Принятый сигнал записывается как функция угла поворота — строится поляризационная эллипс. По отношению максимума к минимуму определяется осевое отношение (AR).

**Метод трёх компонент:**
Для полного описания поляризационного состояния измеряются три линейные компоненты с разными ориентациями. Из них восстанавливается полный поляризационный вектор Стокса.

**Что получают:** Тип поляризации, осевое отношение, угол наклона поляризационного эллипса, кросс-поляризационный уровень XPD.

**Где важно:**
- Спутниковые антенны — AR должен быть ≤ 3 дБ
- ЭМС-измерения — измерения проводятся в двух поляризациях (горизонтальной и вертикальной) раздельно
- Радиолокация — XPD важен для подавления нежелательных отражений

### Измерение коэффициента калибровки (Antenna Factor)

Специфично для измерительных антенн, применяемых при ЭМС-тестировании. Антенна помещается в поле известной напряжённости (генерируемой в ТЕМ-камере или на стандартной площадке по методу ANSI C63.5). Напряжение на выходе антенны измеряется приёмником. Коэффициент калибровки вычисляется как $AF = E/U$.

Результат оформляется как калибровочная кривая $AF(f)$ и вносится в свидетельство о калибровке.

## Основные стандарты и ГОСТы

### Отечественные стандарты

**Антенные измерения:**
- **ГОСТ Р 50414-92** (МЭК 60050-712) — Международный электротехнический словарь. Антенны. Определения терминов.
- **ГОСТ Р 58171-2018** — Антенны. Методы измерений характеристик.
- **ГОСТ 8.508-84** — ГСИ. Метрологические характеристики средств измерений и точностные характеристики средств автоматизации ГСП (общий, но применяется при поверке антенн как СИ).

**ЭМС-измерения (антенны как рабочий инструмент):**
- **ГОСТ Р 51319-2019** (CISPR 16-1-1) — Требования к аппаратуре для измерения радиопомех. Измерительные антенны.
- **ГОСТ Р 51320-2018** (CISPR 16-2-1) — Методы измерения кондуктивных помех.
- **ГОСТ Р 51318.22-2013** (CISPR 22) — Оборудование информационных технологий. Требования к напряжённости поля излучаемых помех.
- **ГОСТ Р 55732-2013** (CISPR 32) — Совместимость технических средств электромагнитная. Оборудование мультимедиа. Требования к излучению.

### Международные стандарты

**IEEE:**
- **IEEE Std 149-1979 (R2003)** — IEEE Standard Test Procedures for Antennas. Базовый стандарт на методы антенных измерений.
- **IEEE Std 1720-2012** — Recommended Practice for Near-Field Antenna Measurements. Детальные требования к измерениям в ближнем поле.

**IEC/CISPR:**
- **CISPR 16-1-4:2019** — Specification for radio disturbance and immunity measuring apparatus. Калибровка измерительных антенн.
- **CISPR 16-1-6:2017** — EMC antenna calibration.

**ETSI:**
- **ETSI TR 102 273** — Методология измерений антенн для телекоммуникационного оборудования. Серия из нескольких частей, охватывает как ближнеполевые, так и дальнеполевые методы.

**Военные и специальные:**
- **MIL-STD-461G** — Requirements for the Control of Electromagnetic Interference. Используется в оборонной промышленности; включает требования к антеннам для измерений излучённых помех.

**Метрология антенн:**
- **ANSI C63.5-2017** — American National Standard for Electromagnetic Compatibility — Radiated Emission Measurements in Electromagnetic Interference (EMI) Control — Calibration and Qualification of Antennas. Основной стандарт на калибровку измерительных антенн в диапазоне 30 МГц — 40 ГГц.
