# Диаграмма направленности антенны

- **Определение в широком смысле** ::
  - Характеристика антенн, показывающая ее направленные свойства.

- **Определение в более узком смысле через КУ антенны или ее КНД** ::
  - Это зависимость коэффициента усиления антенны (или коэффициента направленного действия, если не учитывать потери) от направления ее излучения в заданной плоскости.

- **Что физически характеризует ДН?** ::
  - Направленные свойства антенны.

- **Что количественно показывает ДН?** ::
  - Распределение излучаемой (или принимаемой) антенной мощности в пространстве.

- **Как вычисляется комплексная ДН антенны?** ::
  - Как преобразование Фурье (в общем виде непрерывное) от амплитудно-фазового распределения электромагнитного поля по раскрыву (апертуре) антенны.
  - Следствие из математического определения: Управляя распределением тока по апертуре (амплитудой и фазой возбуждения элементов) можно формировать нужную ДН.
  - Как изменится математическое определение комплексной ДН в случае рассмотрения реальной дискретной линейной антенной решетки? Непрерывное преобразование Фурье заменяется дискретным преобразованием Фурье (т. е. интеграл переходит в сумму).

**Иллюстрация диаграммы направленности** 
ДН антенной решетки из 50 элементов в азимутальной плоскости. На графике присутствуют пики и провалы ДН, а также средний уровень боковых лепестков.

## Основные исследуемые параметры диаграммы направленности

- Ширина главного лепестка ДН
- Уровень боковых лепестков ДН (УБЛ ДН)

### Ширина главного лепестка диаграммы направленности

- **Как определяется?** ::
  - По уровню -3 дБ от максимума (если речь идет про нормированную ДН антенны).

#### Взаимосвязь ширины главного лепестка диаграммы направленности антенны с ее коэффициентом усиления

- Чем уже ДН антенны (чем меньше ширина ГЛ ДН), тем выше коэффициент усиления антенны.

#### Взаимосвязь ширины главного лепестка диаграммы направленности антенны с ее углом обзора

- Чем уже ДН антенны (чем меньше ширина ГЛ ДН), тем уже ее угол обзора.

### Глубина главного нуля разностной диаграммы направленности

- **Определение** ::
  - Значение, равное отношению максимума к глубине нуля между двумя главными лепестками ДН.

### Уровень боковых лепестков диаграммы направленности

- **Аббревиатура** ::
  - УБЛ.

- **Определение** ::
  - Максимум первых лепестков после главного.

- **Что характеризует УБЛ?** ::
  - Мощность, которая тратится в "ненужных направлениях".

## Классификация диаграммы направленности по виду характеристики

- Амплитудная ДН  
- Мощностная ДН  
- Фазовая ДН
- Поляризационная ДН

### Амплитудная диаграмма направленности

- **Определение** ::
  - Нормированная зависимость амплитуды излученного электрического поля от направления в пространстве.

- **Формула амплитудной ДН** ::
  - $$ F_{a}(\theta,\varphi) = \dfrac{|E(\theta,\varphi)|}{|E_{max}|} $$
  - где |E(θ,φ)| — напряжённость излученного электрического поля в направлении (θ, φ)  
  - |E_max| — максимальное значение излученного электрического поля.

### Мощностная диаграмма направленности

- **Определение** ::
  - Нормированная зависимость мощности излучения антенны от направления излучения в пространстве.

- **Формула мощностной ДН** ::
  - $$ F_p(\theta,\varphi)=\dfrac{P(\theta,\varphi)}{P_{max}} $$
  - где P(θ,φ) — мощность, излучаемая в направлении (θ,φ)  
  - P_max — максимальное значение излученной мощности.

#### Соотношение мощностной диаграммы направленности антенны с амплитудной

- **Формула соотношения МДН с АДН** ::
  - Т. к. мощность пропорциональна квадрату амплитуды, получаем: $$ F_p = F_a^2 $$
  - где F_p — мощностная ДН, F_a — амплитудная ДН.

### Фазовая диаграмма направленности

- **Определение** ::
  - Зависимость фазы излучённого электрического поля от направления в пространстве.

- **Что показывает?** ::
  - Насколько фаза поля отличается от некоторого опорного значения при изменении угла наблюдения.

- **Как будет изменяться фаза в идеальной антенне с плоским фазовым фронтом?** ::
  - Фаза не будет меняется с углом.

### Поляризационная диаграмма направленности

- **Определение** ::
  - Зависимость поляризации излучённого поля от направления в пространстве.

- **Что показывает?** ::
  - Как меняется ориентация вектора электрического поля при отклонении от главного направления.
  - Пример: Антенна может излучать линейную поляризацию точно по оси, но эллиптическую — под углом.

## Классификация диаграммы направленности по форме в графическом представлении

- Направленная  
  - Игольчатая  
- Веерная  
- Косеканская  
- Изотропная  
- Тороидальная

### Тороидальная диаграмма направленности

- **Какие антенны имеют данный вид ДН?** ::
  - Штыревая антенна  
  - Дипольная антенна  
  - Щелевая антенна

- **Особенность данного вида ДН** ::
  - Симметричность в азимутальной плоскости  
  - Провал вдоль оси  
  - Иллюстрация тороидальной ДН (существует изображение)

## Классификация диаграммы направленности по виду моноимпульсного сигнала

- Суммарная  
- Разностная

### Суммарная диаграмма направленности

- **Форма** ::
  - Один главный лепесток, направленный на цель по равносигнальному направлению.

- **Какие параметры цели измеряются по СДН?** ::
  - Дальность до цели  
  - Скорость цели

- **Способ запитки элементарных излучателей фазированной антенной решётки для получения СДН** ::
  - Синфазный.

### Разностная диаграмма направленности

- **Форма** ::
  - В направлении на цель (в равносигнальном направлении, там, где у суммарной ДН максимум) у РДН глубокий провал, а по бокам от этого направления — 2 или 4 противофазных лепестка.

- **Для чего используется?** ::
  - Для индикации отклонения цели от равносигнального направления.

- **Принцип работы (взаимосвязь амплитуды, фазы, направления и его угла)** ::
  - Когда цель находится на оси равносигнального направления, сигнал в разностном канале равен 0.  
  - Когда цель отклоняется от равносигнального направления, в разностном канале появляется сигнал, амплитуда которого пропорциональна углу отклонения, а фаза (полярность) указывает направление смещения.

- **Какие действия принимает система по информации в разностном канале?** ::
  - Куда нужно довернуть антенну (отклонить луч ДН).

# Коэффициент направленного действия (КНД)

- **Классическое определение КНД через изотропный излучатель** ::
  - Это отношение интенсивности излучения антенны в заданном направлении к интенсивности излучения изотропного излучателя с той же полной излучаемой мощностью.

- **Более емкое определение КНД через введение понятия усредненной интенсивности излучения** ::
  - Это отношение интенсивности излучения антенны в заданном направлении к усредненной интенсивностью излучения.

- **Понимание КНД антенны через КНД изотропного излучателя (или КНД полуволнового диполя)** ::
  - Это характеристика, показывающая, во сколько раз необходимо увеличить мощность, подаваемую на изотропный излучатель (или полуволновой диполь), чтобы он в заданной точке пространства создавал такое же значение вектора Пойнтинга, как и исследуемая антенна.

- **Определение через физическое понимание** ::
  - Относительная величина, характеризующая степень концентрации электромагнитного поля в заданном направлении, т. е. она характеризует степень выполнения антенной свойств пространственного фильтра.

- **Чем определяется КНД?** ::
  - КНД целиком определяется формой нормированной диаграммы направленности антенны.
  - Следствие: Хочешь высокий КНД → делай узкий главный лепесток ДН; хочешь широкий обзор → КНД будет низким.

- **Единицы измерения** ::
  - Безразмерная величина, которая, однако, записывается в логарифмическом масштабе:  
    - дБи — децибелы относительно изотропного излучателя (наиболее распространено)  
    - дБд — децибелы относительно полуволнового диполя; значение всегда на 2,15 дБ меньше, чем в дБи  
    - дБ — подразумевается дБи, если не указано иное.

- **Чем отличается КНД антенны от его коэффициента усиления антенны?** ::
  - КНД описывает только геометрию излучения, он не учитывает потери, в отличие от КУ.

## Точная формула определения коэффициента направленного действия

- **Точная формула для расчета КНД в произвольном направлении (θ, φ)** ::
  - $$ D = \dfrac{4\pi \cdot U(\theta,\varphi)}{P_{rad}} = \dfrac{4\pi \cdot U(\theta,\varphi)}{\int_0^{\pi}\int_0^{2\pi} U(\theta,\varphi)\sin\theta \, d\theta \, d\varphi} = \dfrac{4\pi \cdot U(\theta,\varphi)}{\int_0^{\pi}\int_0^{2\pi} B_0 F_p(\theta, \varphi) \sin\theta \, d\theta \, d\varphi} $$
  - где U(θ,φ) — интенсивность излучения антенны в направлении (θ,φ)  
  - U_i — интенсивность излучения идеального изотропного излучателя  
  - 4π — полный телесный угол  
  - P_rad — полная излучаемая мощность антенны  
  - B_0 — максимальная интенсивность излучения  
  - F_p(θ, φ) — мощностная диаграмма направленности.

- **Как изменится точная формула для расчета КНД, если потребуется определить КНД в направлении главного лепестка?** ::
  - U(θ, φ) в числителе заменится на U_max, т. е. D = 4π · U_max / ∫∫ U(θ,φ) sinθ dθ dφ.

- **Исходя из точной формулы для расчета КНД, что нужно знать для определения КНД?** ::
  - Необходимо знать трехмерную диаграмму направленности антенны.

- **Причина, по которой данной формулой на практике пользуются редко** ::
  - Необходимость брать двойной интеграл по всей сфере, который аналитически берется лишь для простейших антенн. Для реальных антенн необходимо моделирование.

### Пояснение к выводу точной формулы КНД

- Если бы антенна излучала равномерно во все стороны (как точечный изотропный излучатель), какова была бы интенсивность в каждом направлении? ::
  - Она была бы равна полной излучаемой мощности антенны, делённой на полный телесный угол сферы: U_i = P_rad / 4π.

- Чем отличается излучение реальной антенны от излучения изотропного излучателя? ::
  - Реальная антенна в направлении главного лепестка ДН создаёт интенсивность U(θ,φ), которая выше этого среднего значения.

- Вспоминая определение КНД ::
  - Это отношение реальной интенсивности антенны в направлении (θ,φ) к интенсивности изотропного излучателя с той же полной мощностью излучения.

## Приближенные формулы определения коэффициента направленного действия

- **В чем заключается суть приближения?** ::
  - Вместо вычисления двойного интеграла по всей сфере используется более простая геометрическая оценка.

- **Основной используемый параметр для грубой оценки КНД** ::
  - Ширина главного лепестка диаграммы направленности по уровню -3 дБ в азимутальной и угломестной плоскостях, измеряемая в градусах (θ_1d и θ_2d).

- **Как выбираются плоскости (азимутальная и угломестная)?** ::
  - Они выбираются так, чтобы обе проходили через направление максимума излучения и были перпендикулярны друг другу.

### Приближенные формулы для грубой оценки КНД направленных антенн

- Формула Краусса (наименьшая точность)  
- Формула Эллиота для антенных решеток (средняя точность)  
- Формула Тей и Перейра (наивысшая точность)

#### Формула Краусса

- **Точность данной формулы** ::
  - Наименьшая среди всех.

- **Какую антенну предполагает?** ::
  - Идеальную.

- **Какую оценку дает для антенных решеток?** ::
  - Завышенную.

- **Формула** ::
  - $$ D_{0} = \dfrac{41253}{\theta_{1d} \cdot \theta_{2d}} $$

#### Формула Эллиота для антенных решеток

- **Точность данной формулы** ::
  - Выше, чем у формулы Краусса, но меньше, чем у формулы Тей и Перейра.

- **Что дает меньший числитель в сравнении с формулой Краусса?** ::
  - Он эмпирически учитывает мощность, уходящую в боковые лепестки.

- **Формула** ::
  - $$ D_{0} = \dfrac{32400}{\theta_{1d} \cdot \theta_{2d}} $$

#### Формула Тей и Перейра

- **Точность данной формулы** ::
  - Наивысшая.

- **Что дают квадраты в знаменателе?** ::
  - Меньшую погрешность в случае сильно несимметричного луча.

- **Формула** ::
  - $$ D_{0} = \dfrac{72815}{\theta_{1d}^{2} + \theta_{2d}^{2}} $$

#### Пример грубой оценки КНД направленных антенн по приближенным формулам

Пусть антенна имеет ширину луча θ_1d = 3° и θ_2d = 3° (симметричный узкий луч), тогда:

- Формула Краусса: D_0 ≈ 4584 ≈ 36.6 дБи  
- Формула Эллиота: D_0 ≈ 3600 ≈ 35.6 дБи  
- Формула Тей и Перейра: D_0 ≈ 4045 ≈ 36.1 дБи  
Разброс между формулами — около 1 дБ.

### Приближенные формулы для грубой оценки КНД ненаправленных антенн с тороидальной диаграммой направленности

- **Когда и для каких антенн применяется?** ::
  - Применяется для штыревых, дипольных и щелевых антенн, имеющих тороидальную диаграмму направленности (симметричную в азимутальной плоскости и имеющую провал вдоль оси).

- **Классификация приближенных формул** ::
  - Формула Макдональда  
  - Формула Позара

#### Формула Макдональда

- **Точность** ::
  - Выше при наличии заметных боковых лепестков в тороидальной ДН.

- **Формула** ::
  - $$ D_{0} = \dfrac{101}{\Delta\theta_d - 0{,}0027 \cdot \Delta\theta_d^{2}} $$

#### Формула Позара

- **Точность** ::
  - Выше при слабых боковых лепестках (в идеале — без них).

- **Формула** ::
  - $$ D_{0} = -172{,}4 + 191\sqrt{0{,}818 + \dfrac{1}{\Delta\theta_{d}}} $$

# Интенсивность излучения

- **Определение** ::
  - Скалярная физическая величина, равная средней мощности, переносимой электромагнитной волной через единицу площади, расположенной перпендикулярно направлению распространения волны.

- **Определение через телесный угол** ::
  - Мощность, излучаемая антенной в единичный телесный угол в направлении (θ, φ).

- **Обозначение** ::
  - U(θ, φ), где θ — угол в вертикальной плоскости (0–180°), φ — угол в горизонтальной плоскости (0–360°).

- **Формула через вектор Пойнтинга** ::
  - U(θ,φ) = r² · |S(θ,φ)|

- **Формула через амплитудную ДН** ::
  - U(θ,φ) = B₀ F_a²(θ,φ) (Вт)

- **Формула через мощностную ДН** ::
  - U(θ,φ) = B₀ F_p(θ,φ) (Вт)

- **Отличие физического смысла интенсивности излучения в сравнении с вектором Пойнтинга** ::
  - Интенсивность излучения зависит только от направления и не зависит от расстояния.

- **Как перейти от интенсивности излучения к полной излучаемой мощностью антенны?** ::
  - Проинтегрировать интенсивность излучения по всей сфере.

# Усредненная интенсивность излучения

- **Определение** ::
  - Это интенсивность излучения идеального изотропного излучателя, которая равна отношению полной излучаемой мощности данного идеального изотропного излучателя к полному телесному углу.

- **Формула** ::
  - $$ U_{i} = \dfrac{P_{rad}}{4\pi} $$

# Вектор Пойнтинга

- **Определение** ::
  - Векторная физическая величина, характеризующая плотность потока энергии (мощность, проходящую через единицу площади, перпендикулярной направлению распространения).

- **Физический смысл** ::
  - Вектор Пойнтинга характеризует поток энергии электромагнитного поля.

- **Единица измерения** ::
  - Вт/м².

- **Формула** ::
  - $$ \vec{S} = [\vec{E} \times \vec{H}] $$

- **Как Вектор Пойнтинга направлен относительно векторов E и H?** ::
  - Перпендикулярно.
  - Следствие: Направление вектора Пойнтинга совпадает с направлением распространения электромагнитной волны.

- **Что характеризует комплексный Вектор Пойнтинга?** ::
  - Реактивные поля в ближней зоне (реактивной зоне).  
  - Диаграмму направленности, если взять реальную часть комплексного вектора Пойнтинга.

# Проблематика вектора Пойнтинга

- **Как вектор Пойнтинга связан с расстоянием?** ::
  - Он обратно пропорционален квадрату расстояния.

- **Проблематика вектора Пойнтинга** ::
  - Его использование как характеристики антенны неудобно, поскольку приходится привязываться к конкретной точке пространства.

- **Решение данной проблемы** ::
  - Использование понятия, отвязанного от расстояния — интенсивности излучения.

# Полная излучаемая мощность антенны

- **Определение** ::
  - Мощность, излучаемая антенной во всех направлениях.

- **Формула через интенсивность излучения** ::
  - $$ P_{rad} = \oint U(\theta,\varphi) \, d\Omega = \int_0^{\pi}\int_0^{2\pi} U(\theta,\varphi)\sin\theta \, d\theta \, d\varphi $$

# Коэффициент усиления антенны (КУ)

- **Фундаментальное определение** ::
  - Это отношение интенсивности излучения антенны в заданном направлении к интенсивности излучения изотропного излучателя, к которому подведена та же входная мощность P_in.

- **Определение через КНД** ::
  - Это коэффициент направленного действия антенны с учетом всех потерь.

- **Принципиальное различие КУ и КНД** ::
  - В отличие от КНД, КУ учитывает все потери в антенне: омические потери, потери на рассогласование и т. д.

- **Физический смысл** ::
  - Показывает, во сколько раз антенна создаёт большую интенсивность излучения в направлении максимума по сравнению с изотропным излучателем при той же подводимой мощности. Это главный практический параметр антенны.

- **Обозначение** ::
  - G (в отечественной и зарубежной литературе); реже K (в старой отечественной литературе).

- **Единицы измерения** ::
  - дБи (относительно изотропного излучателя)  
  - дБд (относительно полуволнового диполя; на 2.15 дБ меньше, чем в дБи)

## Формулы

- **Точная формула** ::
  - $$ G(\theta,\varphi) = \dfrac{4\pi \cdot U(\theta,\varphi)}{P_{in}} $$

- **Формула через КНД** ::
  - $$ G = \eta \cdot D $$

- **Полный КПД** ::
  - $$ \eta = \eta_{ref} \cdot \eta_{ohm} $$
  - η_ref = 1 - |Γ|² (КПД по рассогласованию)  
  - η_ohm — КПД по омическим потерям

- **Формула через нормированную ДН** ::
  - $$ G_0 = \dfrac{\eta \cdot 4\pi}{\int_0^{\pi}\int_0^{2\pi} F_0(\theta,\varphi)\sin\theta \, d\theta \, d\varphi} $$

- **Сравнение с КНД** ::
  - G ≤ D всегда. Равенство достигается только у идеальной антенны без потерь (η = 1).

## Методы измерения КУ на практике

### Метод двух антенн

- **Суть** ::
  - Два одинаковых экземпляра исследуемой антенны располагают друг напротив друга и измеряют отношение принятой мощности к переданной.

- **Формула** ::
  - $$ \dfrac{P_r}{P_t} = G^{2}\left(\dfrac{\lambda}{4\pi R}\right)^{2} $$

- **Ограничение** ::
  - Требует двух идентичных экземпляров антенны.

### Метод трёх антенн

- **Суть** ::
  - Используют три разные антенны, проводят три попарных измерения. Получают систему из трёх уравнений и решают её относительно G₁, G₂, G₃.

- **Преимущество** ::
  - Не требует одинаковых антенн.

### Метод эталонной антенны (метод замещения)

- **Суть** ::
  - КУ исследуемой антенны определяется относительно эталонной антенны с известным КУ (обычно стандартный рупор).

- **Формула** ::
  - G_DUT = G_ref + ΔP_dB

### Метод сканера ближнего поля

- **Суть** ::
  - Измеряется трёхмерная ДН через NF-FF преобразование, затем КУ вычисляется численно по точной формуле с учётом КПД антенны.

## Связь КУ с коэффициентом калибровки антенны

- **Коэффициент калибровки (Antenna Factor)** ::
  - AF = E / U

- **Физический смысл** ::
  - Чем выше КУ антенны, тем эффективнее она принимает поле, тем меньше AF.

- **Формула пересчёта (для согласованной антенны, Z₀ = 50 Ом)** ::
  - AF_dB = 20 lg f_MHz - G_dBi - 29.8

- **Применение** ::
  - AF указывается в калибровочном свидетельстве измерительной антенны и используется для пересчёта показаний приёмника в значения напряжённости поля при измерениях ЭМС.
