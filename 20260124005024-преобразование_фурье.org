- Аббревиатура :: FT.
- Исторический аспект:
  - Когда предложен? :: В начале XIX века.
  - Кем предложен? :: Джозефом Фурье (1768–1830), французским математиком и физиком.
- Определение (бытовое) :: Математический процесс декомпозиции сложных функций на элементарные составляющие.
- Аналогия из мира музыки :: Аккорд -- сложная функция, которую можно разложить на элементарные составляющие (отдельные ноты разных частот).
- Иллюстрация физического смысла преобразования Фурье: 
  [[file:attachments/Fourier_transform_time_and_frequency_domains.gif]]
- Что преобразование Фурье делает с сигналами с математической точки зрения :: Преобразование Фурье вычисляет скалярное произведение сигнала с комплексной экспонентой \(e^{-i 2\pi \nu t}\) для разных частот \(\nu\) для оценки степени присутствия в данном сигнале этой самой частоты.
- Что является результатом преобразования Фурье с математической точки зрения для непрерывных сигналов :: Для непрерывных сигналов это функция \(F(\nu)\), которая каждой частоте \(\nu\) сопоставляет свое [[id:424c0bab-1ecf-46b9-92a3-2a99f981459c][комплексное число]].
- Что является результатом преобразования Фурье с математической точки зрения для дискретных сигналов :: Массив (вектор) [[id:424c0bab-1ecf-46b9-92a3-2a99f981459c][комплексных числел]].

- Абревиатура :: DFT.[fc:0]
- Определение :: Математический процесс декомпозиции последовательности из \(N\) цифровых отсчетов на конечное количество (\(N\)) элементарных составляющих (комплексных экспонент).[fc:1]
- Физический смысл (с точки зрения вида времени и спектра) :: Преобразование конечного числа отсчетов сигнала в конечный спектр частот.[fc:2]
- Главное свойство DFT :: Взаимная однозначность (обратимость).[fc:3]
- Как частота взятия отсчетов в исходной последовательности \(x[n]\) влияет на итоговый спектр \(X[k]\) :: Чем реже берутся отсчёты (ниже частота дискретизации), тем /ниже максимальная частота/, которую можно представить в спектре.[fc:4]
- Как длина сигнала влияет на итоговый конечный дискретный спектр? :: Чем короче сигнал, тем труднее отличить друг от друга похожие частоты.[fc:5]
- Структура спектра после дискретного преобразования Фурье ::
  - Частоты распределены равномерно от 0 до частоты дискретизации \(f_s\). 
  - Шаг между частотами равен \(\Delta f = f_s/N\).
  - Вторая половина спектра после частоты \(f_s/2\) содержит [[id:8a137e7c-db85-4bbe-9540-0ad4b59320d3][комплексно сопряженные]] значения первой половины для вещественных сигналов (спектр для реальных сигналов после частоты \(f_s/2\) *отзеркален*).[fc:6]
- Может ли дискретное преобразование Фурье быть применено к [[id:квантование][квантованному]] сигналу? :: С формальной математической точки зрения -- может.[fc:7]
  - Цена в сравнении с неквантованным сигналом :: Большая погрешность, искажающая итоговый спектр.[fc:8]
